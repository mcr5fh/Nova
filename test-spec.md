# Test Specification

Create a simple "Hello World" program in Go.

## Requirements

1. Create a file `hello.go` in the current directory
2. The program should print "Hello, Nova!" when run
3. Add a test file `hello_test.go` that verifies the output
4. The code should be properly formatted with `go fmt`

## Acceptance Criteria

- Running `go run hello.go` should output "Hello, Nova!"
- Running `go test` should pass
- Code should be clean and follow Go conventions
